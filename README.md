# LAN-Based Entrance Examination and Student Profile System for Guidance Services

**Talibon Polytechnic College**

Production-ready Laravel 11 application for entrance exams and student profiling.

## Features
- RBAC (Admin/Guidance + Student)
- Secure Exam Taking with Timer & Auto-save
- Auto Scoring + Manual Grading
- Live Monitoring
- Student Profiles + Bio Notes
- Reports, Analytics, Audit Logs
- LAN Only Deployment

## Setup
1. Clone repo
2. `composer install`
3. Copy .env.example to .env
4. `php artisan migrate --seed`
5. `php artisan serve --host=0.0.0.0`

Access via local IP.
